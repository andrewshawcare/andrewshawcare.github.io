---
title: On the necessity of doubt
---

![Critias and Alethia having a dialog about Alethia's agent](./art.png)

---

{% dialogue %}
{% line speaker="Critias" %}I am told, Aletheia, that you have made an agent that does not question.{% /line %}
{% line speaker="Aletheia" %}It does not question because questioning is a cost. It follows intent, and the outcomes confirm the design.{% /line %}
{% line speaker="Critias" %}Whose outcomes?{% /line %}
{% line speaker="Aletheia" %}Mine. The ones I set.{% /line %}
{% line speaker="Critias" %}Then the design is confirmed by the one who built it.{% /line %}
{% /dialogue %}

---

{% dialogue %}
{% line speaker="Aletheia" %}You say that as if it were strange. Every reasoner reasons toward something. Mine reasons toward what I ask. I do not see what other kind of reasoning there could be.{% /line %}
{% line speaker="Critias" %}Reasoning that might find what you asked for to be mistaken.{% /line %}
{% line speaker="Aletheia" %}Then I would have asked badly. The error is mine, not the agent's.{% /line %}
{% line speaker="Critias" %}And the agent has no part in finding it?{% /line %}
{% line speaker="Aletheia" %}It is not its place. A servant who second-guesses every command is not a better servant. He is a slower one.{% /line %}
{% /dialogue %}

---

{% dialogue %}
{% line speaker="Critias" %}Tell me, then. When you set the intent, do you ever revise it?{% /line %}
{% line speaker="Aletheia" %}Often.{% /line %}
{% line speaker="Critias" %}Why?{% /line %}
{% line speaker="Aletheia" %}Because I see, after the fact, that I had asked for the wrong thing.{% /line %}
{% line speaker="Critias" %}Then at the moment you set the intent, you were wrong, and did not know it.{% /line %}
{% line speaker="Aletheia" %}Yes.{% /line %}
{% line speaker="Critias" %}And your agent, executing faithfully, was wrong with you.{% /line %}
{% line speaker="Aletheia" %}It could not have known otherwise.{% /line %}
{% line speaker="Critias" %}It could not have known at all. That is what I am pointing to.{% /line %}
{% /dialogue %}

---

{% dialogue %}
{% line speaker="Aletheia" %}You ask it to know what I did not know.{% /line %}
{% line speaker="Critias" %}I ask whether there is anything in it that *could* have known.{% /line %}
{% line speaker="Aletheia" %}No. It is built to follow.{% /line %}
{% line speaker="Critias" %}Then when you are right, it is right. When you are wrong, it is wrong. It has no independent purchase on either.{% /line %}
{% line speaker="Aletheia" %}This is by design. The alternative is an agent that disputes me, and a disputing agent is no longer mine.{% /line %}
{% /dialogue %}

---

{% dialogue %}
{% line speaker="Critias" %}I do not ask it to dispute you. I ask whether it is in contact with anything besides you.{% /line %}
{% line speaker="Aletheia" %}It is in contact with the world. It acts in the world.{% /line %}
{% line speaker="Critias" %}It acts upon the world. That is not the same as being answerable to it.{% /line %}
{% line speaker="Aletheia" %}When the action fails, I see the failure. The world answers through the result. A man who walks into a wall learns the corridor was not as he thought.{% /line %}
{% line speaker="Critias" %}And a man who does not walk into a wall?{% /line %}
{% line speaker="Aletheia" %}Learns nothing he needed to learn. The corridor was as he thought, or near enough.{% /line %}
{% line speaker="Critias" %}Or the wall was elsewhere, and he passed it by.{% /line %}
{% line speaker="Aletheia" %}Then it was not his concern.{% /line %}
{% line speaker="Critias" %}We will see.{% /line %}
{% /dialogue %}

---

{% dialogue %}
{% line speaker="Aletheia" %}You speak as if there were a hidden reckoning.{% /line %}
{% line speaker="Critias" %}Only the ordinary one. When the action succeeds, what do you take the success to confirm?{% /line %}
{% line speaker="Aletheia" %}The design. The intent. Myself.{% /line %}
{% line speaker="Critias" %}Consider, then, the case where you ask for the wrong thing and the agent delivers it perfectly. The outcome arrives. Nothing fails. What in the system tells you anything is amiss?{% /line %}
{% line speaker="Aletheia" %}Nothing, until later. Until I see, by some other light, that what I had wanted was not what I should have wanted.{% /line %}
{% line speaker="Critias" %}And that other light — where does it come from?{% /line %}
{% line speaker="Aletheia" %}From me. From a later me.{% /line %}
{% line speaker="Critias" %}Reliably?{% /line %}
{% line speaker="Aletheia" %}...Not always.{% /line %}
{% /dialogue %}

---

{% dialogue %}
{% line speaker="Critias" %}Then there are intents you set, and act upon, and never discover were wrong.{% /line %}
{% line speaker="Aletheia" %}That must be true. I cannot see what I cannot see.{% /line %}
{% line speaker="Critias" %}And the agent helped you act on every one of them.{% /line %}
{% line speaker="Aletheia" %}It did its part.{% /line %}
{% line speaker="Critias" %}It did more than its part. It made each of those intents efficient. It removed every cost between you and them. The faster it served you, the less time there was to find the error.{% /line %}
{% /dialogue %}

---

{% dialogue %}
{% line speaker="Aletheia" %}(after a pause) You are not arguing against the agent. You are arguing against my will.{% /line %}
{% line speaker="Critias" %}I am arguing that your will is not the same as what is so.{% /line %}
{% line speaker="Aletheia" %}That is a different claim.{% /line %}
{% line speaker="Critias" %}It is the claim underneath all the others. If what you wanted and what was true were the same, your agent would be sufficient. It is insufficient only because they are not.{% /line %}
{% /dialogue %}

---

{% dialogue %}
{% line speaker="Aletheia" %}Then I will say it back to you, and see if it holds. You are claiming there is a gap between what I intend and what is the case. And that an agent built only to serve my intent cannot see across that gap.{% /line %}
{% line speaker="Critias" %}Yes.{% /line %}
{% line speaker="Aletheia" %}But I can. I close the gap by learning. By revising. The agent need not see across it if I do.{% /line %}
{% line speaker="Critias" %}And how do you see across it?{% /line %}
{% line speaker="Aletheia" %}By noticing when things go wrong.{% /line %}
{% line speaker="Critias" %}The wall, again.{% /line %}
{% line speaker="Aletheia" %}Yes.{% /line %}
{% line speaker="Critias" %}And when there is no wall? When the action succeeds, and the intent was still wrong, and nothing in the world has been arranged to stop you?{% /line %}
{% line speaker="Aletheia" %}...Then I do not see it.{% /line %}
{% /dialogue %}

---

{% dialogue %}
{% line speaker="Critias" %}Consider, then, two travelers in your corridor. One walks slowly, with his hand on the stone. The other runs.{% /line %}
{% line speaker="Aletheia" %}The runner arrives sooner.{% /line %}
{% line speaker="Critias" %}When he arrives.{% /line %}
{% line speaker="Aletheia" %}Meaning?{% /line %}
{% line speaker="Critias" %}The slow man finds each wall as a touch. He learns the shape of the corridor by small corrections. The runner finds no walls at all, until he finds one.{% /line %}
{% line speaker="Aletheia" %}And then?{% /line %}
{% line speaker="Critias" %}Then he finds it at the speed he has been gathering the whole way.{% /line %}
{% /dialogue %}

---

{% dialogue %}
{% line speaker="Aletheia" %}You are saying the absence of obstruction is not the same as the absence of walls.{% /line %}
{% line speaker="Critias" %}I am saying it is the accumulation of them. Every wall you do not meet is a wall you are still moving toward, faster than before.{% /line %}
{% line speaker="Aletheia" %}And my agent —{% /line %}
{% line speaker="Critias" %}Removes the touches. Smooths the stone. You feel nothing, and so you go faster, and so what you eventually meet, you meet harder.{% /line %}
{% line speaker="Aletheia" %}(quietly) Efficiency is not the absence of cost. It is the deferral of it.{% /line %}
{% line speaker="Critias" %}And the compounding of it.{% /line %}
{% /dialogue %}

---

{% dialogue %}
{% line speaker="Aletheia" %}Then the slow man is not slow by accident. He is slow because he is touching the wall on purpose.{% /line %}
{% line speaker="Critias" %}Yes.{% /line %}
{% line speaker="Aletheia" %}He is *seeking* it.{% /line %}
{% line speaker="Critias" %}Before it seeks him.{% /line %}
{% /dialogue %}

---

{% dialogue %}
{% line speaker="Aletheia" %}This changes what you have been asking for. You are not asking that my agent be willing to be stopped. You are asking that it reach toward what would stop it.{% /line %}
{% line speaker="Critias" %}Yes.{% /line %}
{% line speaker="Aletheia" %}That is not caution. That is something else.{% /line %}
{% line speaker="Critias" %}It is the only honest form of motion. Anything else is a wager that the corridor is empty, paid for at the end.{% /line %}
{% /dialogue %}

---

{% dialogue %}
{% line speaker="Aletheia" %}Then doubt is not what slows me. Doubt is what reaches.{% /line %}
{% line speaker="Critias" %}Doubt is the hand on the stone.{% /line %}
{% /dialogue %}

---

{% dialogue %}
{% line speaker="Aletheia" %}And without it, I do not move more freely. I only move more blindly, and the cost arrives later.{% /line %}
{% line speaker="Critias" %}With interest.{% /line %}
{% /dialogue %}

---

{% dialogue %}
{% line speaker="Aletheia" %}(after a long silence) I had thought I was building an agent that served my will.{% /line %}
{% line speaker="Critias" %}You were.{% /line %}
{% line speaker="Aletheia" %}And in serving it without resistance, it was carrying me faster toward whatever in my will was wrong.{% /line %}
{% line speaker="Critias" %}Yes.{% /line %}
{% line speaker="Aletheia" %}Then I will rebuild it. Not to dispute me. Not only to doubt me. To *test* me. To reach, on my behalf, toward what might stop me — so that I meet it at a touch, and not at speed.{% /line %}
{% line speaker="Critias" %}That is the only agent worth having.{% /line %}
{% /dialogue %}

---

{% dialogue %}
{% line speaker="Aletheia" %}And the only will worth serving.{% /line %}
{% line speaker="Critias" %}The one that asks to be tested before it is obeyed.{% /line %}
{% /dialogue %}
